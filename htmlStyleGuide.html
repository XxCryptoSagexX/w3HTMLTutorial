A Comprehensive list of guidelines and tips to creating good HTML code

  • Always declare document type Example: !DOCTYPE
  • Use lowercase element names
  • Close all HTML elements
  • Use lowercase attribute names
  • Always quote attribute values
  • Always specifiy alt, width, height for images
  • Spaces and Equal Signs: It is best to do "class=" than "class = "
  • Avoid long code lines
  • Avoied blank lines and Indentation without reason
  • Never skip the title element
  • Never omit the body and html elements
  • Never omit the head element
  • Always add the lang attribute
  • Always add meta data charset for proper indexing and character coding.
  • Always set the viewport
  • Use lowercase file names (Totally did not follow this...)
  • Use the appropriate file extention names per document. IE Javascript=".js" HTML=".html or htm" CSS=".css"
