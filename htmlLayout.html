HTML Layout Techniques

There are four different techniques to create multicoloumn layouts. Each technique has its own pros and cons:

  • CSS Framework
  • CSS float property
  • CSS Flexbox
  • CSS Grid

CSS Fameworks

If you want to create your layout fast, you can use a CSS framework, like W3.CSS or Bootstrap.

  • W3Schools W3.CSS Course
  • W3Schools Bootstrap Course

CSS float Property

It is common to do entire web layouts using the CSS float property. Float is easy to learn - you just need to remember how the float and clear properties work. Disadvantages: Floating elements are tied to the document flow, which may harm the flexibility.

CSS Flexbox

Use of flexbox ensures that elements behave predictably when the page layout must accommodate different screen sizes and different display devices

  • CSS Flexbox Example

CSS Grid

The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.
